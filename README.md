### 🏠 House Price Prediction using Linear Regression

## 📌 Overview
This project predicts house prices based on **area (in square feet)** using a simple linear regression model.  
It demonstrates the end‑to‑end machine learning workflow: dataset handling, model training, visualization, evaluation, and interactive prediction.

---

## 🎯 Problem Statement
House prices are influenced by many factors, but area is one of the most significant.  
The goal of this project is to build a regression model that predicts the price of a house given its area, providing a foundation for more advanced real estate analytics.

---

## 🛠 Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, NumPy, scikit‑learn, Matplotlib  
- **Environment:** Jupyter Notebook  

---

## 📂 Dataset
The dataset (`data-houseprice.csv`) contains:
- `area` → Area of the house (sq. feet)  
- `price` → Price of the house (rupees)  

---

## 🚀 How to Run
1. Clone the repository
 
2. Navigate to the folder:
   
   cd House-Price-Prediction
   Install dependencies:
          pip install -r requirements.txt
   
3. Open the notebook:
   
   jupyter notebook HousePricePrediction.ipynb
   
4.Run cells step by step:

 - Import libraries

 - Load dataset

 - Train regression model

 - Visualize scatter plot + regression line

 - Enter area (sq. feet) when prompted → get predicted price

## 📊 Results
Scatter plot of area vs. price with regression line.

Example prediction:
  Enter the area (in sq feet): 2000
  Predicted price: 45,000 rupees
  
Model performance:

  R² score: 0.92 (shows strong fit to data).
